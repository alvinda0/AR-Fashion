# Shirt Models

Letakkan file 3D model untuk kemeja dan kaos di folder ini:

## Format yang Didukung:
- **casual_shirt.glb** - Kemeja kasual
- **formal_shirt.glb** - Kemeja formal  
- **t_shirt.glb** - Kaos polos

## Cara Mendapatkan Model 3D:

### 1. Download dari Website
- **Sketchfab**: https://sketchfab.com/3d-models/clothing
- **TurboSquid**: https://www.turbosquid.com/3d-models/clothing
- **CGTrader**: https://www.cgtrader.com/3d-models/clothing

### 2. Buat Sendiri dengan Software 3D
- **Blender** (Gratis): https://www.blender.org/
- **Maya** (Berbayar)
- **3ds Max** (Berbayar)

### 3. Gunakan AI Generator
- **Meshy**: https://www.meshy.ai/
- **Luma AI**: https://lumalabs.ai/

## Tips Optimasi:
1. Pastikan model memiliki < 10,000 triangles
2. Texture resolution maksimal 1024x1024
3. Export dalam format GLB untuk performa terbaik
4. Pastikan model sudah di-rig untuk body fitting
